# Weather-Predicting Application
A simple and responsive Weather Application built using HTML, CSS, and JavaScript. This app allows users to check real-time weather information for any city, including temperature, humidity, and weather conditions.

🚀 Features
🌍 Search weather by city name
🌡️ Real-time temperature display
💧 Humidity and weather conditions
☁️ Dynamic weather icons
📱 Responsive design

🛠️ Tech Stack
HTML
CSS
JavaScript
Weather API (OpenWeatherMap)
